Setup Instructions

To start tracking your Claude Code usage, add this hook to your Claude Code settings:

{
  "hooks": {
    "onToolUse": "curl -X POST http://localhost:5000/api/log -H 'Content-Type: application/json' -d '{\"type\": \"tool_usage\", \"tool_name\": \"$TOOL_NAME\", \"session_id\": \"$SESSION_ID\"}'"
  }
}

Note: This is a proof-of-concept. Full integration would require a Claude Code extension or MCP server.
